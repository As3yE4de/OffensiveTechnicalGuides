# Hide data inside hidden volume - Technical concepts
A volume, also known as a drive, is a collection of addressable sectors that an Operating System (OS) or application can use for data storage. The sectors in a volume need not be consecutive on a physical storage device; instead, they need to only give the impression that they are, users simply need to open File Explorer and store their data in a volume, they don't need to know the structure of the volume at physical level. Taking advantage of this, the attacker only needs to find a way to make a volume hidden from the view of the average user.
